# LetsWatch Extension Installation

To install this extension for development/testing:

1.  Open Chrome and navigate to `chrome://extensions`.
2.  Enable **Developer mode** (toggle switch in the top right corner).
3.  Click the **Load unpacked** button (top left).
4.  Select this folder: `c:\projects\letswatch\extension`.
5.  The extension "LetsWatch Side Panel" should now appear in your list.

## Usage
1.  Go to any website (e.g., YouTube, Netflix).
2.  Click the **Extensions** puzzle piece icon in Chrome toolbar.
3.  Select **LetsWatch Side Panel**.
4.  It will open in the side panel. You can now join a room and start streaming!
